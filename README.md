# Codeforces Tracker App

Codeforces Tracker App is an Android native application built with Kotlin and XML, based on the MVVM architecture. It is designed to help users keep track of programming contests on Codeforces, a popular platform for competitive programming. The app provides various features and functionalities to enhance the user's experience and keep them up to date with the latest contests.

## Features

1. **Contest Tracking**: The app provides a comprehensive list of upcoming contests, recent contests, and live contests that are currently ongoing. Users can easily stay informed about the contests happening on Codeforces.

2. **User Profile Stalking**: Users can stalk and view the profiles of other Codeforces users. This feature allows users to explore other programmers' profiles and learn from their coding achievements.

3. **Rating Charts and Analysis**: The app offers detailed data analysis of a player's performance, including rating charts, problem-solving statistics, tag-based problem analysis, and submission verdicts represented with pie charts. Users can track their progress over time and analyze their strengths and weaknesses.

4. **Profile Management**: Users can save their Codeforces profiles within the app and easily access them for analysis and tracking. This feature allows users to monitor their progress and evaluate their performance over time.

## Tech Stack

The Codeforces Tracker App is built using the following technologies:

- Kotlin: The primary programming language for developing the Android app.
- XML: Used for creating the app's user interface and layouts.
- MVVM Architecture: Follows the Model-View-ViewModel architectural pattern to separate concerns and improve maintainability.
- Codeforces API: Integrates real-time API data to fetch information about upcoming contests and user profiles.
- MPAndroidChart Library: Used to create rating charts and data visualizations in the app.
- Android Studio: The IDE used for development and testing of the app.
